“Patron AI is a full-stack AI-powered customer support chatbot built with Node.js, Express, MongoDB, and Google Gemini API. It features secure user authentication, dynamic real-time chat, persistent message history, and a clean responsive UI using HTML, CSS, TailwindCSS, and JavaScript.”

# Patron AI - AI Powered Customer Support Chatbot 🚀

Patron AI is a full-stack AI-powered chatbot web application built to simulate customer support interactions using Google Gemini API. It features user authentication, dynamic chat responses, persistent chat history with MongoDB, and a clean, responsive UI.

> Built by Yash Vaswani, Devanand KV, Sania Gade and Neelam Gupta

---

## 📌 Features

- ✨ AI-Powered responses using Google Gemini 2.0
- 🔐 Secure user authentication (signup & login)
- 💬 Interactive real-time chatbot UI
- 🛡️ Persistent chat history storage with MongoDB
- 📱 Responsive and modern frontend design (HTML5, CSS3, TailwindCSS)
- ⚡ Fast and reliable backend (Node.js, Express.js)
- 🧠 Context-aware smart conversations
- 🚪 Logout and session management
- 🌈 Clean and user-friendly design for smooth experience

---

## 🚀 Tech Stack

| Frontend | Backend | Database | AI Integration |
|:--------|:--------|:---------|:---------------|
| HTML5, CSS3, TailwindCSS, JavaScript | Node.js, Express.js | MongoDB | Google Gemini API |

---

## 🛠️ Project Architecture

- Frontend: HTML, CSS, TailwindCSS for styling; JavaScript for dynamic UI
- Backend: Node.js + Express for API routes and server
- Database: MongoDB Atlas/local for user and chat storage
- AI: Google Gemini API integration to generate smart, dynamic responses
- Hosting: Ready for deployment on platforms like Render, Vercel, Netlify

## 📄 How to Run Locally

1. Clone this repository:
   bash
   git clone    https://github.com/VaswaniYash/Patron-AI
   cd patron-ai-chatbot

   	2.	Install backend dependencies: npm install
   	3.	Set up .env file: GEMINI_API_KEY=your_google_gemini_api_key_here
   	4.	Start the server: node server.js
   	5.	Open your browser and go to: http://localhost:5500
